4) Why isn’t it possible to see the first five rows of a table? 
>In RDBMS table is not a file,every row is a file.
Rows of table not store sequentially.
Rows of the table are scattered(fragmented) all over the database(DB) server harddisk(HD) (common for all RDBMS). 
So thats why it is not possible to see first 5 rows of a table.
